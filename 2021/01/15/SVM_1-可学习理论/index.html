SVM_1：可学习理论

By Dongqi Shen

SVM_1: 从 Hoeffding不等式 到 VC维

不知道这么说正不正确，私以为SVM在机器学习中有着完美的数学理论解释，就像贝叶斯定理，拥有一套完整的数学理论框架。深层神经网络在当下很是流行，但它的缺点是缺乏可解释性。虽然SVM目前已很少应用，但是依赖它完备的数学理论使得它依然在机器学习中占有一席之地。况且，谁能知道在未来的哪个阶段，也许以SVM为代表的传统机器学习算法又能迎来新的一春呢？
我打算把SVM的读书笔记写成一个系列，具体有多少章，现在的我也答不上来。在我看来，SVM是入门机器学习最好的学习路径，因为他有很强的数学理论证明，这些都是机器学习的基本知识。本章笔记的命名就很有理论性，可能很多机器学习算法从业者都没有听说个这两个概念，因为大多数教程都直接从算法入手，而忽略了机器学习的基本理论支撑，这一章就是从Hoeffding出发，为什么机器学习是有效的。

引入Hoeffding不等式

通常我们所知的机器学习就是给你一个样本数据集，然后选择模型，训练，调参，最终得到一个训练好的结果（模型），用这个结果去预测未知的数据。可能绝大多人认为这是一气呵成，理所当然的，但是有没有想过这个因果关系存在的原因是什么？为什么从样本数据集上得出的结果能够表示全局数据。可学习性就是来回答这样一个问题。
学习的目的就是找到目标函数（模型）f，但是我们并不知道这个目标函数，我们已知的只有有限的样本集，并且希望从这有限的样本集中找到目标函数f，或者找不到（99.9%都找不到）而去找一个非常接近f的函数，这是机器学习最大的难点，从有限的数据中找无限数据的规律。
我们先从一个简单的例子入手，假设有个垃圾桶，桶内放置了红色和绿色的玻璃珠，μ（未知）表示红色珠子的概率，相对应1-μ表示拿到绿色珠子的概率。现在我们从垃圾桶中有放回得拿取一些珠子（样本），用v（已知）表示样本中红色珠子的比例。那么，我们能从v告诉了我们哪些关于μ的信息呢？为了用v来表示μ，引入一个简单的边界，Hoeffding Inequality。
\(\begin{equation}P[|v-μ| > \epsilon]\le 2e^{-2ε^{2}N} \ for \ any\ ε > 0\end{equation}\)
这里的P[.]表示一件事的可能性，这里取决与我们选取的随机样本。用文字表述这个不等式就是随着样本数量N的增长，v偏离μ以指数级的速度收敛在容忍值ε内。如果为了让v去逼近μ，可以设置ε为非常小的数，这样子的话就需要一个很大的样本数量N。关于Hoeffding不等式是来自大数定理的一个结论，可以看作是客观存在的整理，我们只是拿来使用而不去证明它。

可学习性边界

机器学习的目的就是找到一个objective function f: x->y. 我们先定义一个假设集H，样本数据集D。从假设集H中选取一个特定的假设h，首先计算在数据集D上的样本误差（in-sample error） \(E_{in}(h) = \frac{1}{N}\sum_{n=1}^{N}{[h(X_n)\neq f(X_n)]}\) 其次是样本外误差(out-of-sample error) \(E_{out}(h) = P[h(x) \neq f(x)]\) 把他们带入上述的Hoeffding不等是，可以重写如下 \(P[|E_{in}(h)-E_{out}(h)| > ε]\leq 2e^{-2ε^{2}N} \ for \ any\ ε > 0\) 其中的h代表一个特定的假设集，但是在假设空间上存在着M个假设集，我们的目标是把这M个假设集都考虑进去。为了说明，我们先引入一个显而易见成立的不等式。假设$B_1,B_2,…,B_M$是一系列事件，那么 \(P[B_1\ or\ B_2\ or\ ... B_M] \leq P[B_1] + P[B_2] + ...P[B_M]\) 如果每个事件之间有很大的重合，右边明显比左边大，如果每一个事件都不重合，则等号成立。将这个不等式应用于M个假设，则有 \(P[|E_{in}(g) - E_{out}(g)| > ε] \leq P[|E_{in}(h_1) - E_{out}(h_1)| > ε \\ or \ |E_{in}(h_2) - E_{out}(h_2)| > ε \ or \ ...\\ or \ |E_{in}(h_M) - E_{out}(h_M)|> ε ] \leq \sum_{m=1}^{M}{P|E_{in}(h_m) - E_{out}(h_m)| > ε}]\) 根据前面的不等式，我们可以用M个边界来替代，由此可以得出 \(P[|E_{in}(g) - E_{out}(g)| > ε] \leq 2Me^{-2ε^{2}N}\) 这个不等式就是关于整个假设集的上界，其中g表示我们最终选择的假设。可以看出，这是一个相当松弛的边界，不等式的左边是一个假设，而不等式的右边是关于整个假设集上界，并且只有在M是有限的时候这个不等式才有意义。

泛化理论

泛化更通俗一点的解释就是通用。一般的机器学习任务，都是从一个数据集上训练，找出合适的数学模型，并用这个模型去预测接下来出现的无穷无尽的新的数据，这是模型的泛化。但是这里的泛化理论指的是从多个、不同的数据集训练出来的模型，是否都能达到这样的效果。试想一下，样本数据是从一个特定分布（要拟合的函数f）上获取的，因此样本数据的分布存在不确定性，比方说全国人民身高，从北方获取的数据，或者从南方获取的数据训练出来的模型显然是不同的，但是这个利用这个带有偏见的模型去预测全国人民的身高，它的误差是否在一个可控范围内（确定边界），这是需要数学去证明的。 可学习性边界利用到了Hoeffding不等式，再一次写出来，如下 \(P[|E_{in}(g) - E_{out}(g)| > ε] \leq 2Me^{-2ε^{2}N}\) 可以把它右边的值用δ表示，通过等式变换得出ε的表达式 \(ε = \sqrt{\frac{1}{2N}\ln(\frac{2M}{δ}})\) 再把P里面的的表达式取出来，可得 \(E_{in} \leq E_{out} + \sqrt{\frac{1}{2N}\ln(\frac{2M}{δ}})\) 现在还是回到M的问题。M代表的是假设集的数量，但是有时候它很难被测量，并且经常是无限的，因此下一步要做的就是用一个有限的值代替M，这会涉及到很多名词的定义。 Dichotomies $x_{1}, x_{2}, …, x_{N}$是从分布X上选取的N个样本点，二分是指假设集能在这N个样本点的样本集上产生所有的分类。无论N个点怎么取，只要存在N个点满足这个要求，就算是二分。 growth function 增长函数指的是假设集对于N个点最多能产生多少种划分，记为$m_{H}(N)$。 break point 如果k个点不能被假设集二分，则k是假设集H的一个断点 这些都是在二分类基础上的定义，多分类也适用，但是证明更加复杂。根据二分的定义，N个点所产生的所有可能的分类有$2^N$种，增长函数满足$m_{H}(N) \leq 2^N$。很容易得出，在二维坐标系中，二维感知机的增长函数是3。可以这么解释，存在三个点，感知机能产生所有8个种类的划分，但是对于四个点，无论怎么摆放，最多只能产生14种可能的划分，因此，$m_{H}(3)=8,m_{H}(4)=14$。可以看出$m_{H} < 2^k$。断点的意义在于，一旦找到最小的一个断点，对于任何一个大的样本集数量N，增长函数的能够在多项式上收敛，这是最终要的一点。这样我们可以用一个多项式商界的增长函数代替N。证明有点复杂，我放弃了（哈哈哈）。

VC维

终于到了最后一个知识点了。 VC维 对于一个假设集H，能使$m_{H} = 2^N$成立的最大的N的值,用$d_{vc}$表示。 显而易见，vc维+1是最小的断点，满足$k = d_{vc} + 1$，如果对于所有的N，都满足$m_{H} = 2^N$，那么这个假设集的vc维是无限的。因此，前面的未经证明的结论，可以用下面的公式表示 \(m_{H}(N) \leq N^{d_{vc}} + 1\) 因此，可以通过vc维的多项式表示增长函数的上界。我们最终的目的是希望代替可学习型边界中的M，因为M的值往往是无穷大的。增长函数不能直接代替M，但是通过对增长函数的一些修改就能达到这个目的。这个过程也存在严谨的数学理论推导，我看了几遍，没记住，还是只记住结论吧，最后得出的公式是： \(E_{out}\leq E_{in}(g) + \sqrt{\frac{1}{N}\ln \frac{4m_{H}(2N)}{δ}}\) 简单分析一下右边根号内的情况，可以分为两个函数$y_{1} = x$，$y_2 = \ln m_{H}(x)$，其中$m_{H}$是多项式，显然随着x增加，$y_{1}$比$y_2{2}$增长得更快，在坐标轴上表示的话，$y_{1}$一直在$y_{2}$上面。因此根号内的值随着N的增大，一直在缩小。这就意味着，如果有足够多的数据，有限假设集中的每一个拥有有限vc维的假设，都能很好得拟合目标函数。到这里就解释了为什么机器学习算法需要很多数据的理论证明。

尾声

其实这个结论在实际应用上很少用到，因为这是一个相当宽松的上界，但它的意义就在于从数学理论方面解释了机器学习的可学习性。整个证明过程其实就是找有限边界的过程，并且要让这个边界能够支撑一个理论，那就是从有限的数据集上找到一个函数用来逼近产生这个数据集的函数，这个边界就是误差。不过在整个过程中都没有提及假设集的选择问题，这就涉及到了另外一个理论bias and variance。在寻找边界的过程中，首先使用到了假设集的数量，但是这个值很大并且可能是无穷大的，接着就希望找一个函数来代替这个值，这就有了增长函数。通过一系列的数学证明，增长函数是一个多项式，这是非常重要的一个特性。用增长函数替代假设集数量后（有一些变化），得出一个重要的结论，就是只要有足够多的数据，就能够尽可能逼近函数f。
